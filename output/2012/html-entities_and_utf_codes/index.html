HTML-сущности и их UTF-коды

  • 11 Марта 2012
  • •
  • Комментарии
  • •
  • css

Генерируемое содержимое псевдоклассов :before и :after очень удобно использовать для создания иконок, буллитов, кавычек и в прочих декоративных целях.

Но свойство content:'x' отображает HTML-сущности так, как они записаны в CSS, не заменяет их на специальные символы. Например, напишем в CSS content:'&rarr;', получим в браузере «&rarr;» вместо «→». Поэтому писать спецальные символы в content нужно 4-значными UTF-кодами (UTF-16 hex), экранированными обратным слэшем.

Пример CSS-кода:

p.readmore a:after {
    content: '\203A'; /* выведет › */
    position: absolute;
    right: 0.8em;
    top: 4px;
    display: block;
}
