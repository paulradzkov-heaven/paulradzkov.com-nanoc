Как отключить javascript в Google Chrome

  • 27 Ноября 2012
  • •
  • Комментарии
  • •
  • Chrome

В настройках Chrome Developer Tools запрятано много полезных, не сразу заметных, опций. Самые востребованные — это отключение яваскрипта и отключение кэша.

Чтобы попасть в настройки, откроем Developer Tools (F12) и нажмем иконку шестеренки в правом нижнем углу.

Обратите внимание на опцию «Show paint rectangles» — она позволяет увидеть, какие области перерисовывает браузер в ответ на какие-либо события или в результате работы скриптов. Полезно для профилирования и оптимизации.

Можно поменять строку User Agent и притвориться Интернет Эксплорером, Фаерфоксом, iOS или Андроид устройством. Можно задать строку вручную.

«Device metrics» позволяют протестировать вёрстку на мобильных разрешениях, в том числе с изменением font scale фактора и имитацией тач-событий. Без всяких дополнительных плагинов.

Хочу заметить, что это не эмуляция других браузеров. Баги и особенности ренедера не воспроизводятся. Зато можно посмотреть, что возвращает сервер для разных браузеров.

Из горячих клавиш для меня оказались полезными Ctrl + Shift + F — поиск по всем ресурсам страницы, и F2 — редактировать как html.
